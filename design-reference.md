- lg: 18px / line-height 1.7

### Heading（見出し）
- xs: 16px / line-height 1.5
- sm: 18px / line-height 1.5
- md: 20px / line-height 1.4
- lg: 24px / line-height 1.4
- xl: 28px / line-height 1.4

### Button
- sm: 12px / line-height 1.5
- md: 14px / line-height 1.5
- lg: 16px / line-height 1.5

## 適用方針
- 本文は16px以上を基本とする
- 見出しは20-24pxを使用
- ボタンは16px以上
- 補助テキストでも14px以上を維持


## Semantic Colors

### Text
- Text Main: #32353a（基本テキスト）
- Text Sub: #686a6d（副次的テキスト）
- Text Link: #3959cc（リンク）
- Text Placeholder: #96989a

### Background
- Background White: #ffffff（基本背景）
- Background Blue: #f0f2fc（薄い青背景）
- Background Pink: #fcf0f5
- Background Green: #e9faf4

### Primitive Colors（Blue系）
- Blue 100: #f0f2fc（最も薄い）
- Blue 200: #cfd7f3
- Blue 300: 中間
- Blue 500: #3959cc（メインブルー）
- Blue 600-700: ダーク

## 適用方針
- 背景は白(#ffffff)を基本とする
- カード背景は薄いブルー(#f0f2fc)
- プライマリカラーは#3959cc（LINC Blue）
- テキストは#32353a（濃いグレー）
